Homelab Ingress Architecture
AdGuard DNS + Traefik routing + Step-CA trust chain + service segmentation + observability
DNS Layer
AdGuard is now explicitly shown as authoritative resolver for internal hostnames and wildcard/local records.
Ingress Layer
Traefik handles TLS termination, routing, middleware chains, fallback, and metrics exposure.
Trust Layer
Step-CA provides internal cert issuance; root CA distribution to devices is mandatory for clean TLS UX.
Ops Layer
Structured logs + Prometheus endpoint provide baseline observability for incident response and tuning.
